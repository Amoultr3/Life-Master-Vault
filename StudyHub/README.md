# StudyHub

StudyHub is a medical learning system for nursing school and future advanced study.

Core sections:
- Library
- Chapter Notes
- Study Maps
- Remediation
- Uploads
- Search

Philosophy:
- Chapter notes live with chapters.
- Notes can still be linked and searched anywhere.
- Study maps can function as review checklists or concept maps.
- Uploaded lecture materials can be linked to notes, tags, chapters, and maps.
- Search should find everything related to a term, no matter where it started.
